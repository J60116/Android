# Androidアプリ開発（練習）
## 機能一覧
* 2種類の猫から1匹選んで画面遷移
* フリック操作により、猫を撫でたりツッコミさせる
* 撫でた回数とツッコミ回数をカウント
* 日本語と英語に対応

（今後修正したいところ: 撫でる操作の有効範囲を狭くする、ドラック操作の実装）

## 製作期間
2024年1月~

## 使用技術
* 言語: Java
* 開発環境: Android Studio
